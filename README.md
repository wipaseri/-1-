# 🎬 サブスク迷子救済ガチャ

> 「何見ようかな…」を終わりにする、ストリーミングサービス向け作品ランダム提案ツール

サブスクに登録しているのに、何を見るか決められない。そんな「サブスク迷子」状態を解決するWebアプリです。登録中のサービスと今の気分を選ぶだけで、TMDb APIからおすすめ作品をランダムに1つ提案します。

## ✨ 機能

| 機能               | 説明                                                                         |
| ------------------ | ---------------------------------------------------------------------------- |
| 複数サブスク対応   | Amazon Prime / Netflix / U-NEXT / Hulu / Disney+ / dアニメストア             |
| 体力別フィルタ     | 「映画2時間いける」か「ドラマ1話だけ」かで作品タイプを切り替え               |
| 豊富なジャンル     | **18種類のジャンル**から選択（コメディ・ドラマ・アクション・SF・ホラーなど） |
| 気分タイプ選択     | ロマンチック / 興奮・サスペンス / 思考的 / リラックス / 冒険 / ダーク        |
| フィルタオプション | 人気順 / 高評価のみ / 最新作 / ランダム                                      |
| ランダム選出       | 条件に合った作品からランダムに1つをピック                                    |
| 充実した情報表示   | ポスター・タイトル・公開年・評価・あらすじ表示                               |

## 🚀 クイックスタート

### 1. リポジトリをクローン

```bash
git clone https://github.com/<your-username>/tonight-watch.git
cd tonight-watch
```

### 2. TMDb APIキーを取得

1. [TMDb 公式サイト](https://www.themoviedb.org/) でアカウントを作成
2. [API設定ページ](https://www.themoviedb.org/settings/api) へ移動
3. **v3 API key** を生成・コピー

### 3. APIキーを設定

`index.html` の以下の行を編集します。

```javascript
// 変更前
const TMDB_API_KEY = "YOUR_TMDB_API_KEY_HERE";

// 変更後（取得したキーに置き換え）
const TMDB_API_KEY = "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx";
```

### 4. ブラウザで開く

```bash
# ダブルクリック or ブラウザにドラッグ&ドロップ
open index.html   # macOS
start index.html  # Windows
```

サーバー不要 — `index.html` をブラウザで直接開くだけで動作します。

## �🎮 使い方

1. **登録しているサブスクを選択**（複数選択可能）
   - Amazon Prime、Netflix、U-NEXTなど
2. **体力レベルを選択**
   - 「体力あり」→ 映画（2時間フル視聴）
   - 「体力なし」→ TV・アニメ（1話だけ）

3. **ジャンルを選択**（18種類から選択）
   - コメディ、ドラマ、アクション、SF、ホラー等

4. **気分タイプを選択**
   - ロマンチック 💕 / 興奮 ⚡ / 思考的 🤔 / リラックス 😌 等

5. **フィルタを選択**
   - 人気作品 / 高評価のみ / 最新作 / ランダム

6. **「ガチャを回す！」ボタンをクリック**

7. 表示された作品を今夜見る 🍿

「もう一度回す」ボタンで別の作品を何度でも再抽選できます。

## ❓ トラブルシューティング

### 「作品が見つかりませんでした」

- 選択したサブスクとジャンルの組み合わせで配信作品がない場合があります
- 複数のサブスクを選択するか、ジャンルを変えてお試しください

### ガチャボタンを押してもエラーになる

- `TMDB_API_KEY` に有効なキーが設定されているか確認してください
- ブラウザのコンソール（F12）でエラー内容を確認してください
- TMDb側のAPIレート制限（10req/秒）に引っかかっている場合は少し待ってください

### あらすじが英語で表示される

- TMDbのデータベースに日本語訳がない作品の場合があります（仕様）

## � セキュリティに関する注意

⚠️ **本番環境でのAPIキー管理**：

このアプリはクライアント側にTMDb APIキーを埋め込んでいます。本番環境での運用では以下の対応を推奨しています：

- **Cloudflare 環境変数の使用**（推奨）
  - Cloudflare Pages のプロジェクト設定で環境変数を設定
  - JavaScriptで環境変数経由でアクセス

- **バックエンド API の実装**（より安全）
  - Cloudflare Workers でTMDb APIをプロキシ化
  - クライアント側は自社エンドポイント経由でアクセス

詳細は [CLAUDE.md](CLAUDE.md) の「デプロイメント」セクションを参照してください。
